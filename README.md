# KezaIO 🖤✦

> Your private AI life advisor — built by [Jiamini Innovations Ltd](https://distrihub.co.ke)

KezaIO is a mobile-first personal AI companion that knows you deeply and helps you grow intentionally. Unlike social media or public platforms, KezaIO is entirely for you — your thoughts, your goals, your career, your patterns.

---

## Modules

| Module | Description |
|---|---|
| 🌙 Mirror | Private journaling with AI pattern analysis |
| 🔥 Habits | Daily habit tracking with AI coaching |
| 🧠 Clarity | Second brain — brain dump, AI organises |
| 💙 Pulse | Mood & energy tracking with personal dashboard |
| 🎯 Goals | Life vision holding with daily AI connections |
| 💼 Career | AI job matching, CV tailoring, ATS PDF, domain email |
| 📚 Books | Personalised book recommendations + nearby stores |
| ✉️ Email | Custom domain email module |

---

## Tech Stack

- **Mobile** — Flutter (Android + iOS)
- **Backend** — Django REST Framework
- **Database** — PostgreSQL (Supabase)
- **AI** — Self-hosted open-source LLM (Mistral 7B / LLaMA 3)
- **Payments** — M-Pesa (Kenya) + Stripe (international)

---

## Monetisation

Freemium model — core features free, premium AI features gated.

- **Free** — journaling, habits, mood tracking, 10 AI interactions/month
- **Premium** — unlimited AI, job hunting, CV tailoring, domain email, all themes
- **Pricing** — KES 499/month · KES 4,499/year
